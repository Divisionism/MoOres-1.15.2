--------------------------------------------------------------
How to download and compile the mod yourself
--------------------------------------------------------------

==============================================================
1) Download the files from the repo
==============================================================
2) Extract the ZIP file
==============================================================
3) Open Command Prompt
==============================================================
4) Change the directory to the file you have just unzipped
==============================================================
5) Type gradlew build
==============================================================
6) Wait for it to finish
==============================================================
7) Now go to build/libs
==============================================================
8) There should be a jar file. That's the mod file
==============================================================
